URL重定向漏洞防范

1.什么是Url重定向漏洞

URL重定向（URL redirection，或称网址重定向或网域名称转址），是指当使用者浏览某个网址时，将他导向到另一个网址的技术。

网站接受用户输入的链接，跳转到一个攻击者控制的网站，可能导致跳转过去的用户被精心设置的钓鱼页面骗走自己的个人信息和登录口令。

2.如何防范Url重定向漏洞

服务端配置跳转白名单或域名白名单，只对合法的URL做跳转

错误写法:

String backUrl = request.getParameter("backUrl");
response.sendRedirect(backUrl);

正确写法(只跳转白名单中的地址):

String[] urlArray = ["http://www.xiangyun.com","http://www.baidu.com"];
String backUrl = request.getParameter("backUrl");
try{
    int dest = Integer.parseInt(backUrl);
    if((dest >= 0) && (dest <= urlArray.length - 1)){
        response.sendRedirect(urlArray[dest]);
    }
} catch(NumberFormatException e){
    response.setCharacterEncoding("utf-8");
    response.setContentType("text/html");
    response.getWriter().write("即将跳转到其他不可信任的网址：" + backUrl + "</br>请确认是否继续</br><a href=" + backUrl + ">继续</a><button>返回</button>);
}
