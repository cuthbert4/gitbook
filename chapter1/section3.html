非关系型数据库

如何防范非关系型数据库Sql注入

涉及到非关系型数据库mongdb在查询时不能使用拼接sql的方式，需要绑定参数进行查询，跟关系型数据库的预编译类似

错误写法(直接拼接用户输入的查询条件)

String username = request.getParameter("username");
MongoCollection<Document> mc = mongoClient.getDatabase("MyDB").getCollection("account");
BasicDBObject Query = new BasicDBObject();
Query.put("$where","this.username==\""+username+"\"");
FindIterable<Document> find = mc.find(Query);

正确写法(参数绑定)

String username = request.getParameter("username");
MongoCollection<Document> mc = mongoClient.getDatabase("MyDB").getCollection("account");
BasicDBObject Query = new BasicDBObject();
Query.put("username",new BasicDBObject("$eq",username));
FindIterable<Document> find = mc.find(Query);
